# **Zeta-Physics-Engine**

3D Physics Engine (Mechanics) created to be used by a 3D graphics library.

![Logo](yourmom_png.png)

___

## Example Compilation:
* On linux: `make -f makefile`
* With WSL: `make win -f makefile`
* With MinGW: `mingw32-make.exe win`
